# Artifact Sketcher

تبدیل تصاویر اشیای باستانی (مانند کوزه و سفال) به اسکچ‌های سیاه‌وسفید پررنگ با استفاده از پردازش تصویر در OpenCV.

## توضیح پروژه
این اسکریپت با استفاده از افزایش کنتراست (CLAHE)، حذف نویز، آستانه‌گذاری تطبیقی و عملیات مورفولوژیک، لبه‌ها و ساختار اصلی شیء را استخراج کرده و یک خروجی اسکچ واضح و مناسب مستندسازی تولید می‌کند.

## کاربردها
- مستندسازی آثار باستانی  
- آماده‌سازی تصاویر برای تحلیل فرم و کانتور  
- تولید اسکچ پژوهشی بدون استفاده از Deep Learning  

## پیش‌نیازها
Python 3.x  
opencv-python  
numpy  
matplotlib  

## نحوه اجرا
نام فایل تصویر را در متغیر `input_filename` قرار دهید و اسکریپت را اجرا کنید.  
خروجی با نام `bold_ancient_sketch.jpg` ذخیره خواهد شد.
